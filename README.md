# video_silce

### 文件结构

##### video_silce/
##### │
##### ├── main.py                    // 程序入口，通过该文件执行完整过程
##### ├── config.py                  // 配置文件（记录API密钥、视频路径等）
##### ├── requirements.txt           // 依赖库列表及其版本
##### ├── README.md                  // 项目说明
##### │
##### ├── src/                       // 主要代码文件
##### │   ├── __init__.py            // 空文件，把src文件当成一个python包
##### │   ├── video_processor.py     // 视频处理模块，对视频进行分割、剪辑等视频操作
##### │   ├── audio_processor.py     // 音频提取模块，把输入的视频转换为音频
##### │   ├── speech_to_text.py      // 音频转文字模块，利用模型讲音频转为带有时间戳的文本
##### │   ├── text_analyzer.py       // 文本分析模块（或包含评分模块），利用ai把转录的文本进行处理、评分、筛选
##### │   ├── prompts.py             // ai的提示词模板
##### │   ├──utils.py               // 工具函数，如时间的格式化显式之类的小功能，暂时可以用不上
##### │   └──………………                 // 后续改进新增的文件   
##### │ 
##### ├── data/                      // 数据目录
#####     ├── input_videos/         // 存放输入的原始视频
#####     ├── processed_audio/      // 存放视频转换成的音频文件
#####     ├── transcripts/          // 存放音频转录的文本文件
#####     ├── output_videos/        // 存放生成的短视频
#####     └──……………………………………        // 后续改进新增的文件
